# React Admin Dashboard → Jaspr (Dart)

This project is a Dart + Jaspr rewrite inspired by the original React admin dashboard:
https://github.com/ed-roh/react-admin-dashboard

## What was converted
- Dashboard shell layout (sidebar + topbar)
- Main pages/routing:
  - Dashboard
  - Team
  - Contacts
  - Invoices
  - Bar / Line / Pie
  - FAQ
  - Form
- Styled UI using `web/styles.css`

## Tech
- Dart
- Jaspr
- jaspr_router

## Run locally
```bash
jaspr serve
```
Then open the local URL printed in terminal.

## Build
```bash
jaspr build
```
